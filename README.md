# Project Paeon

사업자등록번호를 통해 정확한 상호명을 추려내는 사이트

## Dev

```bash
> python -m venv venv
> source venv/bin/activate
> pip install -r requirements.txt
> python manage.py runserver
```

## How to use

1. 사업자등록번호를 입력합니다.
2. 상호명으로 유추되는 회사명 입력 (여러개를 동시에 집어넣어도 됩니다.)
3. 검색 후 목록 보기
